Instellingen

v1.5.9

API-server (iPhone)

Vul hier het adres in van de Shift Happens API-server (meestal poort 3847), bijvoorbeeld http://192.168.1.10:3847 — niet de Yesplan- of ticket-URL; die staan onder Yesplan/Itix. Leeg laten bij opslaan behoudt de huidige server-URL. Alleen nodig voor de iPhone-app.

Thema

Grotere scrollbalk en grotere tikgebieden voor gebruik op aanraakschermen (bijv. Raspberry Pi-kiosk).

Zaalvolgorde

Sleep de zalen om de volgorde aan te passen. Klik op het oog om een zaal te verbergen in de zaal-dropdown.

Technische opties per zaal

Kies per zaal welke opties (balletvloer, vleugel, orkestbak, zaalplattegrond) getoond worden. Stel dit in op basis van wat je in Yesplan per zaal ziet.

Yesplan

Bepaal welke Yesplan-organisatie het dashboard gebruikt

Organisatie 1

Haalt zalen op voor zaalvolgorde en header

Organisatie 2

Haalt zalen op voor zaalvolgorde en header
Vul per organisatie Base URL en API Key in, klik op "Laad Zalen"

Itix

Voer het voorvoegsel van de beheer-URL voor de zaalplattegrond in, eindigend vóór het uitvoerings-id (Yesplan-event-id). Voorbeeld: https://tickets.jouworganisatie.nl/beheer/zaalplattegrond/uitvoeringinfo

Priva

Updates (desktop)

Alleen actief in de geïnstalleerde app. Met npm start (ontwikkeling) is er geen automatische update.
